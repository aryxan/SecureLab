# SecureLab

A collection of cybersecurity tools written in **C++** for learning, experimentation, and practical security research.

This repository is part of my journey in cybersecurity where I build small but useful tools to understand **network security, cryptography, and system protection**.

---

## Tools Included

### Port Scanner
A lightweight TCP port scanner that identifies open ports on a target machine.

Features:
- Fast port scanning
- Detects open/closed ports
- Useful for network auditing

---

### Packet Sniffer
Captures and analyzes network packets to understand network traffic.

Features:
- Packet capture from network interface
- Displays source and destination IP
- Basic packet inspection

---

### File Integrity Monitor
Detects unauthorized modifications to files using cryptographic hashing.

Features:
- File hashing
- Change detection
- Tampering alerts

---

### Encryption Tool
Encrypts and decrypts files securely.

Features:
- File encryption
- Password-based protection
- Secure data storage

---

### Log Analyzer
Analyzes system or server logs to detect suspicious activity.

Features:
- Detect repeated login attempts
- Identify suspicious IP addresses
- Simple pattern detection

---
